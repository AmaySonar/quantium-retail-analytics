# Data

The raw data files used in this project are provided by Quantium as 
part of their Virtual Experience Programme and are not included in 
this repository in accordance with their data usage terms.

## How to get the data

1. Visit the Quantium Virtual Experience Programme on Forage:
   https://www.theforage.com/simulations/quantium/data-analytics-rqkb
2. Complete Task 1 to download the data files:
   - QVI_transaction_data.xlsx
   - QVI_purchase_behaviour.xlsx
3. Place them in this folder before running the notebooks

## Sample data

A 50-row anonymised sample is available in the `sample/` folder 
for reference purposes only.
